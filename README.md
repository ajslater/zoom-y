# zoom-y

An AppleScript for maximizing the vertical 'zoom' of windows to the display height.

## Install

1. Run the `./compile.sh` script.
1. Bind the compiled `zoom-y.scpt` to a hotkey with the app of your choice.


## Binding

- [BetterTouchTool](https://folivora.ai/) is a good option, but a paid app.
- Creating an Automator Service is built in to MacOS as detailed on [Stack Exchange](https://apple.stackexchange.com/questions/175215/how-do-i-assign-a-keyboard-shortcut-to-an-applescript-i-wrote).
